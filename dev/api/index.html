JacobiElliptic.Pi — Function
Pi(n, m) -> Any

$\Pi(n;\varphi \,|\,m)=\int_{0}^{1 }{\frac{1}{1-nt^{2}}}{\frac{dt}{\sqrt{\left(1-mt^{2}\right)\left(1-t^{2}\right)}}}.$

Returns the complete elliptic integral of the third kind.
